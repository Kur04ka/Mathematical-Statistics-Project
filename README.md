# Данное приложение - проект по математической статистике. 
# В качестве двух аргументов принимает два массива со случайными величинами X и Y, притом одной СВ X соответствует одна СВ Y.

Программа выполняет следующие действия:
1) Task1 - Составляет интервальный вариационный ряд для СВ X, рассчитывает наименьшую и набольшую СВ X,
Рассчитывает длину интервала по правилу Стерджеса. 
2) Task3 - Рассчитывает выборочное среднее, исправленную выборочную диспредсию и исправленное среднеквадратическое отклонение для СВ X и СВ Y.
3) Task4 - Рассчитывает вероятности попадания СВ X в интервал
4) Task6_7 - Рассчитывает выборочный коэффициент корреляции и наблюдаемое значение критерия при n = 100. 

Уровень значимости в данном проекте по условию был 0.05, соответсвенно доверительная вероятность равна 1 - 0.05 = 0.95
